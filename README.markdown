# Logging.js

   Logging library for Node.js based on the java.util.logging library.

``` js
var logging = require('logging-js'), logger = logging.get(module)
	.configure(logging.BasicConfiguration);

logger.info('Hello world');
```  
		  
   

## Features
   * Logging
   * Levels
   * Handlers
   * Formatters
